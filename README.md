# Bestie Birthday

A cinematic birthday memory page built with React and Vite.

## Features

- Animated birthday intro and story sections
- Smooth scrolling with Lenis and GSAP
- Motion effects with Framer Motion
- Responsive photo gallery
- Background birthday music with an on/off control

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Tech Stack

- React
- Vite
- GSAP
- Framer Motion
- Lenis
- Tailwind CSS
- Lucide React
